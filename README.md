# How To Use

## Basic Tools
- clear_figure
  - clear matplotlib.pylab figure

## IOTools
- read_csv
  - args
    - fname -> string
  - read csv as dataframe(Pandas)
- read_image
  - args
    - fname -> string
    - asnumpy -> bool: read image as numpy
  - read color image
- save_image
  - args
    - fname
  - save plt figure
- save_object
  - args
    - obj -> list, dict
    - fname -> string
  - save list or dict
